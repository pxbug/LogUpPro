## 🚀 快速开始

### 系统要求

- iOS 15.0+
- Xcode 14.0+
- Swift 5.5+

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/pxbug/LogUpPro.git
```

2. 打开项目
```bash
cd LogUpPro
open LogUp\ Pro.xcodeproj
```

3. 运行项目
- 选择目标设备（iPhone/iPad）
- 点击运行按钮或按下 `Cmd + R`

## 🛠 技术栈

- **SwiftUI** - 用户界面框架
- **Combine** - 响应式编程
- **URLSession** - 网络请求
- **UserDefaults** - 数据持久化
- **async/await** - 异步编程
